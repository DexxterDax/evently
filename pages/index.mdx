# @rbxts/evently

## About

Evently is a lightweight and type-safe event management library built specifically for Roblox TypeScript projects. It provides developers with a streamlined way to define, handle, and trigger events while leveraging TypeScript's strong typing for better reliability and maintainability.

Key Features:
- Type-safe event handling to reduce runtime errors.
- Flexible API for dynamic event management.
- Easy integration into existing Roblox TypeScript projects.

Whether you're building a game or a tool, Evently helps keep your code clean and robust.
